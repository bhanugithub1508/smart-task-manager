# smart-task-manager
Smart Task Manager is a frontend web application built using HTML, CSS, and JavaScript that allows users to create, track, and manage tasks efficiently. It supports task completion, deletion, filtering, and uses browser LocalStorage for data persistence with a responsive user interface.

## Description
A simple and responsive task management web application that allows users to manage daily tasks efficiently.

## Features
- Add, edit, delete tasks
- Mark tasks as completed
- Filter tasks (All, Completed, Pending)
- Persistent storage using browser LocalStorage
- Responsive design for mobile and desktop

## Technologies Used

- CSS
- JavaScript
